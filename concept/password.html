Shoutkey

Shoutkey is a URL shortener based off of dictionary words. For example a shoutkey with key "blue" that redirects to "google.com" means that you can access google by going to "shoutkey.com/blue"
Shoutkey uses only dictionary words to provide a memorable and easy for the user to share and access URLs, but this also makes it susceptible to dictionary attacks. Try to find the hidden link using the dictionary in the concept below!
