Password-Protected (Limited Attempts) Shoutkey

One way to make Shoutkey more secure is to require a password, increasing the time of a dictionary attack from O(n) to O(n^2) for dictionary of size n!
In addition, we can limit the number of password attempts. This decreases the probability of finding a password for a given key from 1 to (max_attempts)/n.
